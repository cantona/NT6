想要拜黃藥師，首先要到陸乘風那兒實習一段時間。
由于來住的阿貓阿狗太多，一向清靜的歸雲莊不堪其擾。在向黃藥師申請後，並由其指點，在門口設立了一個九宮桃花陣。
九宮桃花陣的破陣方向如下
進入九宮桃花陣後，揀起地上的桃花，指令如  get 3 taohua
____________________
|     |      |     |
|  2  |   9  |  4  |
|_____|______|_____|
|     |      |     |
|  7  |   5  |  3  |
|_____|______|_____|
|     |      |     |
|  6  |   1  |  8  |  
|_____|______|_____|
然後按照 4 3 8 1 6 7 2 9 5的順序將地上的taohua都揀起來，一枝不剩。（也可以不按這個順序走，但是容易暈。）
最後你站在5這個位置上，丟下5枝桃花。
走一步,站在3的位置上,丟3支桃花。接著走到8，丟8枝。
接著1 6 7 2 9 4 分別按數字的多少來丟。
（桃花陣中忽然發出一陣“軋軋”的聲音，隨後現出一條道路，你趕忙走了出去。）
好了，如果這樣你都破不了陣，那就-f吧。
