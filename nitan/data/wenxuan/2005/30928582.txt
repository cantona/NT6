如果不知道怎麼寫計劃，請看下面這個例子以供參考。
scheme edit        
REPEAT            
dazuo 100:RETRY   
yun recover
LOOP
0
0
scheme edit--- 開始制訂計劃內容。
0
REPEAT&LOOP---REPEAT 表示下面的一段一直到 LOOP 之間的指令需要進行循
環執行，注意：一份計劃中所有 REPEAT 命令必須有 LOOP 命令與之匹配，
REPEAT 命令最多可以嵌套 12 層。(注意大寫)
0
dazuo 100:RETRY---冒號 : 後面的 RETRY 表示如果失敗了就重複嘗試，直到成功為止。
當然你也設定為 dazuo 100:sleep 表示在執行命令失敗的時候選擇執行 sleep 命令。
或者，lian dodge 5:lian sword 5，當dodge不能練了以後練習sword。
0
 scheme star---展開計劃，必須在安全區。一般門派休息室或自己的房子裏都可以計劃。
如果像丐幫那樣沒有房子又沒有休息室可以去盤龍計劃。
盤龍-揚州廣場開始{#6 n;e;panlong;n;n;push;n;n;n;n;open door;w;}
0
halt---中止計劃。
0
另外，像yun recover,yun regenerate這樣的命令可以做成alias,
避免重複寫計劃時可能出現的輸入錯誤。
0
lian xxx；du xxx；也可以做成alias。
如:alias dd1 du book 10；alias ll1 lian dodge 10,
這樣，可以直接在alias裏修改計劃，不用重寫計劃那麼麻煩。
0
